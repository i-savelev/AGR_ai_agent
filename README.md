# AGR AI Agent

Этот репозиторий нужен для помощи в подготовке цифровых информационных моделей по требованиям АГР.

## Назначение

Проект используется как рабочая база знаний и набор вспомогательных материалов для анализа, подготовки и проверки данных, связанных с ЦИМ АГР. Репозиторий объединяет исходные нормативные документы, преобразованные машиночитаемые представления, примеры IFC-файлов и служебные утилиты для конвертации материалов в удобный для агента вид.

## Текущая структура

- `data/` — исходные документы и подготовленные производные артефакты, включая Markdown-представления и CSV-таблицы по приложениям требований.
- `ids/` — IDS-файлы с формализованными требованиями и ограничениями для проверки моделей.
- `sample_ifc/` — примеры IFC-файлов ЦИМ АГР.
- `utils/` — утилиты для конвертации и нормализации документов в Markdown, CSV и `docx`.
- `xml/` — XML-файлы ведомостей и связанных структур данных.

## Как используется проект

1. Исходные документы по требованиям АГР собираются в репозитории.
2. С помощью `utils` документы конвертируются в машиночитаемые форматы для дальнейшего анализа.
3. Подготовленные артефакты используются для извлечения требований, построения базы знаний и проверки состава данных ЦИМ.
